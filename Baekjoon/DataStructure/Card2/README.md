카드2(2164)
===
## 개요
+ 1부터 N까지 숫자가 적혀있는 N개의 카드를 순서에 따라 바닥에 버릴때, 가장 마지막에 남는 카드를 구하세요.
## 규칙
+ 다음과 같은 규칙 순서로 카드를 버립니다.
    1. 제일 위에있는 카드를 버립니다.
    2. 제일 위에있는 카드를 제일 마지막으로 옮깁니다.
    3. 위 순서대로 카드가 1장 남을때까지 반복합니다.
+ 카드의수 N은 $1 \le N \le 500,000$을 만족합니다.
## 입력 및 출력
+ 첫번째 줄에 N이 주어집니다.
+ 마지막으로 남게되는 카드를 출력하세요.
